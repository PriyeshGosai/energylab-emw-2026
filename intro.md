# Energy Modelling Workshop 2026

Developed for **EnergyLab**, Energy Modelling Workshop, Vientiane, Laos, July 2026.

This book is a hands-on introduction to energy system modelling with
[PyPSA](https://pypsa.org). It works through five exercises that build up from a
single network to capacity expansion, regional interconnection, and the
management of hydropower dams and cascades.

## Exercises

| # | Exercise | Focus |
|---|----------|-------|
| 1 | Building a PyPSA Network | Constructing and solving a network |
| 2 | Performing a capacity expansion model | Investment optimisation |
| 3 | Regional connection | Linking multiple regions |
| 4 | Dam Management | Modelling hydropower storage |
| 5 | Cascades | Chained hydropower systems |

## Using the material

Each exercise is a Jupyter notebook that reads its input data from an
accompanying Excel workbook. The notebooks in this book are shown with their
saved outputs. To run them yourself, install PyPSA and a solver, then open the
notebooks from the [repository](https://github.com/PriyeshGosai/energylab-emw-2026).
